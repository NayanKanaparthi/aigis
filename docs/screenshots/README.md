# README screenshots

The README references three images at the paths below. Save the screenshots
you captured (the three you pasted into the v2.1 release session) into this
directory with the exact filenames listed.

| Filename | What it shows |
|---|---|
| `setup.png` | Terminal: `aigis init cursor` + the success message |
| `agent-flow.png` | Cursor chat panel: agent reading .cursorrules, identifying triggers, invoking aigis as tool calls |
| `verify.png` | Terminal: `aigis verify input-validation --auto . --jurisdiction eu` output with PASS marks + cross-framework citations |

If a filename mismatches, the README image embed will render as a broken
image link — fix by renaming the file to match.

The screenshots are referenced from `README.md` (root) using relative
paths: `docs/screenshots/<name>.png`.
